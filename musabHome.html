HTML

HTML (HyperText Markup Language) is the standard language used to create and structure content on the web. It defines the structure of web pages using elements like headings, paragraphs, images, links, and more.

CSS

CSS (Cascading Style Sheets) is used to style and layout web pages. It controls colors, fonts, spacing, and layout of the web page.

JavaScript

JavaScript (JS) is a programming language used to create dynamic and interactive web pages. It allows you to manipulate content, create animations, and handle user interactions.
